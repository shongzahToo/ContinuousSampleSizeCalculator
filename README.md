# Sample Size Calculator

This repository contains a simple web-based application to calculate the required sample size for statistical tests. The application is built using Vue.js and Bootstrap for the frontend, and includes features for calculating sample sizes both with and without Bonferroni adjustments.

## Inputs

- **Confidence Level**: Specify the confidence level for the test.
- **Power**: Set the power of the test.
- **Minimum Detectable Effect (MDE)**: Define the smallest effect size that you want to detect.
- **Number of Tails**: Choose between a one-tailed or two-tailed test.
- **Number of Test Variants**: Input the number of variants in the test.
- **Mean Value**: Provide the mean value of the control group.
- **Standard Deviation**: Enter the standard deviation of the control group.
- **Daily Traffic (optional)**: Optionally, input the daily traffic to estimate the test duration.

## Libraries Used

- **Vue.js**: A progressive JavaScript framework for building user interfaces.
- **Bootstrap**: A CSS framework for developing responsive and mobile-first websites.
- **Numeral.js**: A JavaScript library for formatting and manipulating numbers.
